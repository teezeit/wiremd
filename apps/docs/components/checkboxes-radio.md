# Checkboxes & Radio

## Checkboxes

Standard Markdown task list syntax renders as checkboxes.

::: demo
- [ ] Email notifications
- [x] SMS alerts
- [ ] Weekly digest
- [x] Product updates

:::

## In a Form

::: demo
## Preferences

- [x] Send me product updates
- [ ] Send me marketing emails
- [x] Enable two-factor authentication
- [ ] Make my profile public

[Save Preferences]*

:::

## Radio Buttons

Use `( )` and `(x)` for radio groups.

::: demo
Subscription plan

- ( ) Free — 1 user, 5 projects
- (x) Pro — 10 users, unlimited projects
- ( ) Enterprise — unlimited everything

:::

## Inline Options

::: demo
Notify me via

- (x) Email
- ( ) SMS
- ( ) Push notification

:::

## Syntax

```
- [ ] Unchecked checkbox
- [x] Checked checkbox

- ( ) Unselected radio
- (x) Selected radio
```
