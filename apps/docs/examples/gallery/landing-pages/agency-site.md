**Style Variations:**
- `sketch` - Casual, creative agency vibe
- `clean` - Professional, corporate aesthetic
- `wireframe` - Content structure focus
- `tailwind` - Modern, colorful branding
- `material` - Polished, layered design
- `brutal` - Bold, distinctive identity
- `none` - Semantic markup for custom CSS

**Generate this example:**
```bash
wiremd agency-site.md --style sketch
wiremd agency-site.md --style brutal -o agency-brutal.html
wiremd agency-site.md --style clean -o agency-clean.html
```
