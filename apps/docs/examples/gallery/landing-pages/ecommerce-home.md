**Style Variations:**
- `sketch` - Quick prototype for stakeholders
- `clean` - Modern, professional e-commerce
- `wireframe` - Focus on content structure
- `tailwind` - Contemporary retail aesthetic
- `material` - Google Shopping style
- `brutal` - Bold, distinctive brand
- `none` - Semantic markup for frameworks

**Generate this example:**
```bash
wiremd ecommerce-home.md --style sketch
wiremd ecommerce-home.md --style clean -o ecommerce-clean.html
wiremd ecommerce-home.md --style material -o ecommerce-material.html
```
